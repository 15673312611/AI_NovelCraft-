## 💡 使用说明

### 查看章节列表
1. 进入小说详情页
2. 切换到"章节"标签页
3. 在列表中可以看到：
   - 章节序号和标题
   - **正文内容预览**（前 100 字）
   - 字数和状态
4. 鼠标悬停在正文预览上可以看到更多内容

### 预览章节
1. 在章节列表中点击"预览"按钮
2. 查看章节的完整信息：
   - 基本信息（序号、标题、字数等）
   - 章节摘要（如果有）
   - **完整的章节正文**（可滚动查看）
   - 章节备注（如果有）
3. 可以直接点击"编辑"按钮跳转到编辑

### 编辑章节
1. 在章节列表中点击"编辑"按钮
2. 在弹出的表单中编辑：
   - **标题和副标题**
   - **章节摘要**
   - **章节正文**（12 行文本域，可输入大量内容）
   - **状态、字数、公开性**
3. 点击"保存"按钮提交
4. 系统会自动计算字数（如果有内容）
5. 保存后自动刷新列表

### 正文预览截断
```typescript
render: (content: string) => {
  if (!content) {
    return <span style={{ color: 'var(--text-tertiary)' }}>暂无内容</span>
  }
  // 显示前100个字符
  const preview = content.length > 100 ? content.slice(0, 100) + '...' : content
  return (
    <span style={{
      whiteSpace: 'nowrap',
      overflow: 'hidden',
      textOverflow: 'ellipsis',
    }}>
      {preview}
    </span>
  )
}
```
