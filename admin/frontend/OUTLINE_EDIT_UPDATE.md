### 2. 章纲编辑功能 ⭐

#### 列表式编辑关键情节点
- ✅ 每个情节点独立输入框
- ✅ 支持添加新情节点（点击"+ 添加情节点"按钮）
- ✅ 支持删除情节点（点击 × 按钮）
- ✅ 实时更新列表
- ✅ 保存时自动转换为 JSON 格式

#### 可编辑字段
1. **方向**：文本域（2 行）
2. **关键情节点**：动态列表
   - 每个情节点一个输入框
   - 可添加、删除
   - 自动编号（情节点 1、情节点 2...）
3. **伏笔动作**：下拉选择
   - NONE：无伏笔
   - PLANT：埋伏笔
   - REFERENCE：引用伏笔
   - DEEPEN：深化伏笔
   - RESOLVE：解决伏笔
4. **伏笔详情**：文本域（3 行）
5. **副线情节**：文本域（2 行）

## 💡 使用说明

### 查看章纲
1. 进入小说详情页
2. 切换到"章纲"标签页
3. 点击任意章纲的"查看"按钮
4. 查看完整的章纲信息（情节点以列表形式展示）

### 编辑章纲
1. 进入小说详情页
2. 切换到"章纲"标签页
3. 点击要编辑的章纲的"编辑"按钮
4. 在弹出的表单中编辑内容：
   - **编辑情节点**：直接在输入框中修改
   - **添加情节点**：点击"+ 添加情节点"按钮
   - **删除情节点**：点击情节点右侧的 × 按钮
   - **修改伏笔**：选择伏笔动作，填写伏笔详情
5. 点击"保存"按钮提交修改

### 编辑小说
1. 进入小说详情页
2. 点击顶部的"编辑小说"按钮
3. 当前显示提示信息（功能开发中）

## 🔧 技术实现

### 关键情节点处理
```typescript
// 解析 JSON 字符串为数组
const plotPoints = JSON.parse(outline.keyPlotPoints)

// 编辑时维护数组状态
const [plotPointsList, setPlotPointsList] = useState<string[]>([])

// 添加情节点
const addPlotPoint = () => {
  setPlotPointsList([...plotPointsList, ''])
}

// 更新情节点
const updatePlotPoint = (index: number, value: string) => {
  const newList = [...plotPointsList]
  newList[index] = value
  setPlotPointsList(newList)
}

// 删除情节点
const removePlotPoint = (index: number) => {
  setPlotPointsList(plotPointsList.filter((_, i) => i !== index))
}

// 保存时转换为 JSON
const keyPlotPoints = JSON.stringify(plotPointsList)
```
