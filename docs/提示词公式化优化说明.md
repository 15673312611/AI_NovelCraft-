# 提示词公式化优化说明

## 优化时间
2024年12月11日 13:00

## 核心问题诊断

### 之前的问题
1. **指令分散**：有"卷首三章要求"、"核心原则"、"单章原则"，AI抓不住重点
2. **缺乏公式**：只说"要有转折"，但没告诉AI什么时候、怎么设计
3. **无执行标准**：没有明确的检查标准，AI可以自由发挥

### 优化策略
- ✅ 合并分散的指令
- ✅ 引入公式化指导
- ✅ 加入检查标准
- ✅ 明确优先级

## 核心设计公式

### 1. 冲突升级公式（必须遵守）

```
情绪冲突（吵架/冷战）< 利益冲突（财产/职位/名誉）< 生死冲突（健康/陷害/危机）
```

**分段要求**：
- **前1/3章节**：可以有情绪冲突，但必须在第3章前引入利益或生死层面的赌注
- **中1/3章节**：冲突必须升级到利益层面，涉及实质性得失
- **后1/3章节**：冲突必须达到生死或决定性转折层面

**示例对比**：

❌ **错误**（纯情绪冲突）：
```
[冲突] 姜月初 vs 沈修谨的本能维护。赌注：姜月初对婚姻的最后一点幻想。
```
→ 赌注只是"幻想"，属于情绪层面

✅ **正确**（利益冲突）：
```
[冲突] 姜月初 vs 沈修谨+沈氏集团。争夺焦点：价值数亿的专利归属权。赌注：她三年的研究成果和未来事业。
```
→ 赌注是专利和事业，属于利益层面

### 2. 转折植入公式（打破线性）

```
每3-5章必须出现1次"意料之外"的信息或事件
公式：【预期走向A】→【突然出现信息B】→【被迫转向C】
```

**转折类型**：
- 新信息揭露
- 第三方介入
- 意外事件
- 身份反转
- 隐藏能力暴露

**示例对比**：

❌ **错误**（完全线性）：
```
第1章：家宴冲突
第2章：回房解释
第3章：翻出旧物
第4章：早餐对抗
第5章：找律师
```
→ 完全在预期内，没有任何转折

✅ **正确**（有转折）：
```
第1章：家宴冲突
第2章：回房解释
第3章：【转折】意外发现丈夫公司的新药专利，竟然是基于她三年前的研究
第4章：秘密调查专利侵权证据
第5章：联系国际专利律师团队
```
→ 第3章出现意外信息，从"婚姻问题"升级为"知识产权盗窃"

### 3. 能力展示公式（建立人设）

```
第1-3章内必须完成：
【遇到专业问题】→【主角用能力解决】→【旁人震惊/受益】
```

**禁止**：
- 旁白说明
- 他人转述
- 只展示道具不展示使用过程

**要求**：
- 必须有具体的对手/难题
- 主角当场用能力碾压或化解

**示例对比**：

❌ **错误**（只展示道具）：
```
[事件] 姜月初从床底拖出一个尘封的箱子，里面是她大学时期获奖的证书和一份厚厚的、关于癌症特效药的初始研究构想。
[看点] 金手指初现：女主顶级的科研天赋通过具体的、被束之高阁的研究计划书展现。
```
→ 只是展示了道具（计划书），没有展示她用能力做了什么

✅ **正确**（展示使用过程）：
```
[事件] 婆婆突然晕倒，众人慌乱。姜月初上前检查瞳孔和呼吸，冷静判断："急性换气过度综合征，不是心脏问题。"她指挥家庭医生按她的方法处理，婆婆很快苏醒。
[看点] 金手指初现：女主用顶级医学素养，在混乱中完成技术性"打脸"，瞬间立住人设。
```
→ 有问题（婆婆晕倒）→ 用能力解决（医学判断）→ 旁人震惊（众人看她的眼神变了）

### 4. Show, Don't Tell（最高优先级）

**禁止词**：
- "她想"
- "她心中"
- "她感到"
- "她决定"
- "她意识到"

**替换为**：
- 具体动作（删除/拨号/撕毁/转身）
- 对话
- 表情
- 停顿
- 矛盾行为

**示例对比**：

❌ **错误**：
```
[收束] 姜月初在内心对自己说："沈修谨，我们结束了。"这是她人格觉醒的瞬间。
```

✅ **正确**：
```
[收束] 姜月初没有回应，只是默默删除了他的联系方式，然后拨通了律师的电话。
```

## 标签优先级重排

### 之前的问题
标签没有优先级，AI不知道哪个最重要

### 优化后的优先级

1. **[冲突]**（最重要） - 必须有实质赌注
2. **[看点]**（核心吸引力） - 必须可视化
3. **[事件]** - 服务于看点
4. **[结果]** - 必须有实质性变化
5. **[收束]** - 严禁内心独白

### 每个标签的检查标准

#### [冲突] 检查标准
```
格式：对立双方 + 争夺焦点 + 实质赌注（财产/职位/健康/秘密等）
检查：如果赌注只是"情绪"或"面子"，必须升级到利益/生死层面。
```

#### [看点] 检查标准
```
必须是具体可视化的行为/场景，禁止抽象描述。
检查：能否用一个画面/动作描述？如果不能，重新设计。
```

## 最终提示词结构

```
【角色与任务】
【输入信息】
【本卷目标】
【卷内节奏建议】

【核心设计公式】← 新增，合并了之前的三个分散章节
  1. 冲突升级公式（必须遵守）
  2. 转折植入公式（打破线性）
  3. 能力展示公式（建立人设）
  4. Show, Don't Tell（最高优先级）

【冲突设计与人物智商保护】
【结尾风格】
【爽点与反击方式建议】

【keyPlotPoints 标签规范】← 重新组织，按优先级排序
  1. [冲突]（最重要）+ 检查标准
  2. [看点]（核心吸引力）+ 检查标准
  3. [事件]
  4. [结果]
  5. [收束]
  可选标签：[承接]、[推进]、[钩子]

【伏笔字段说明】
【JSON 输出规范】
```

## 预期效果

使用公式化提示词后，生成的章纲应该：

### 1. 冲突有实质赌注
- ❌ 不再是："赌注：姜月初对婚姻的最后一点幻想"
- ✅ 而是："赌注：价值数亿的专利归属权+未来事业"

### 2. 有意外转折
- ❌ 不再是：第1-10章完全线性（家宴→解释→翻旧物→找律师）
- ✅ 而是：每3-5章出现1次意外（发现专利盗窃/第三方介入/身份反转）

### 3. 能力有展示过程
- ❌ 不再是：只展示道具（翻出研究计划书）
- ✅ 而是：用能力解决问题（用医学知识救人/识破造假/推翻权威）

### 4. 无内心独白
- ❌ 不再是："她在内心对自己说..."
- ✅ 而是：只写可观察的行为（删除联系方式/拨通电话/撕毁文件）

## 关键改进点总结

| 维度 | 优化前 | 优化后 |
|------|--------|--------|
| **指令组织** | 分散在3个章节 | 合并为1个"核心设计公式" |
| **执行标准** | 只说"要有转折" | 给出公式：每3-5章1次 |
| **冲突要求** | 只说"优先利益冲突" | 给出分段要求：前1/3可情绪，中1/3必须利益，后1/3必须生死 |
| **能力展示** | 只说"通过场景展示" | 给出公式：遇到问题→用能力解决→旁人震惊 |
| **标签优先级** | 无优先级 | 按重要性排序：冲突>看点>事件>结果>收束 |
| **检查标准** | 无 | 每个标签都有检查标准 |

---

**优化核心**：从"描述性指导"升级为"公式化约束"
**预期效果**：AI能明确知道什么时候、怎么做、如何检查
