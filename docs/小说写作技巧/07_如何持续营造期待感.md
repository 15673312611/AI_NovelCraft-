# 如何持续营造期待感

**来源：** 七猫中文网 - 编辑：烟草

## 期待感的定义

期待感，指的是读者在阅读过程中，所产生的对后续剧情拥有强烈兴趣，急迫看到内容的感觉。

## 一、期待感的来源

### 期待感的来源就是读者的YY感

**YY的普遍性：**
- 无论是看书，还是看电影，又或者是打游戏，都会有YY
- 看书的后续剧情、看电影的结尾、打游戏爆出某件神装
- 包括去做一件事，跟一个美女发生邂逅，跟帅哥的一次偶遇
- 脑袋里面出现的对后续剧情的脑补就是YY
- 当然你会开始期待盼望你跟这个美女/帅哥所发生点什么，人们对此开始期盼

## 二、如何营造期待感

### 拿目前新媒体比较火的几个类型来说：

#### 1. 高手下山的几个师娘/师姐

**期待点设置：**
- 各种各样的形象，跟主角不同的暧昧关系
- 同时主角还拥有多个好看的美女未婚妻
- 总会让人期待，跟这些美女发生些什么
- 主角还拥有医术，玄学，等各种相当牛逼的技能
- 装逼自然要在美女面前装逼，让她对你心意

**期待感来源：**
- 女角色的塑造和她的最终关系，都会让读者想要去一探究竟

#### 2. 战神归来

**期待点设置：**
- 十年前，家破人亡
- 十年后，我带着十万大军归来
- 十年前你杀我全家，十年后我回来报仇了
- 十年前我跟丧家之犬一样，你让我滚
- 十年后我全天下最牛逼，求着我回归家族继承家产

**期待感来源：**
- 报仇/装逼的过程，将会是读者期待发生的东西

#### 3. 小人物崛起

**期待点设置：**
- 大多数都是遭人背叛
- 获得传承变得不太一样了
- 有机会回去狠狠的打脸，改变命运

### 如何营造这些类型的期待感

#### 首先就是人设：
- 师姐、师娘的人设，哪怕是标签化
- 需要在读者的脑海中形成一个固定标签
- 这个师娘，是个大胸的类型
- 这个师姐，是长腿高冷御姐型
- 不同的标签，会在读者脑海中植入不同的印象
- 有了印象，自然会期待与她之间发生什么

#### 战神类型的期待感：
- 来源于仇恨矛盾的塑造
- 深仇大恨，杀父之仇
- 主角的情绪感官，调动起读者的情绪
- 从而调动起读者的期待值

## 三、如何满足期待感

### 如何爽起来

**满足期待感的步骤：**
1. **设定目标** - 拿下师娘/师姐
2. **完成目标** - 拿下师娘/师姐
3. **设定方案** - 快速高效的拿下师娘/师姐
4. **用事件填充** - 所做的事情

## 四、所有的一切，都是为了满足期待感

### 期待感的核心地位：
- 铺垫、设置、人设等东西，都是为了提取读者的期待
- 将期待最大化，可以有效的提高一本书的留存

### 提升期待感的技巧：

#### 1. 尽可能少写无关紧要的剧情
- 单刀直入
- 用小高潮衔接的方法去创作
- **一章一高潮，一章一打脸，全程无尿点**

#### 2. 用大量的丰富的剧情去完善整本书
- 如果创作出现瓶颈
- 可以在第一天写下第二天的剧情
- 每一章200字，详细记录发生什么

### 期待感的效果：
- 读者如果一直对一个故事产生期待
- 那么这本书各方面的数据，一定不会太差

## 核心要点

1. **YY感** - 期待感的来源就是读者的YY感
2. **人设标签化** - 给配角明确的标签，在读者脑海中形成印象
3. **仇恨矛盾** - 通过深仇大恨调动读者情绪
4. **一章一爽** - 一章一高潮，一章一打脸，全程无尿点
5. **删减废话** - 尽可能少写无关紧要的剧情
6. **提前规划** - 提前写下第二天的剧情大纲，每章200字
7. **持续期待** - 让读者一直对故事产生期待，数据就不会差
