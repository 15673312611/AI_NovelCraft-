## 💡 核心要点速记

### 最重要的3个技巧
1. **主线清晰** - 故事的骨架，开篇单刀直入
2. **代入感强** - 情感共鸣，不是情节相似
3. **人设立体** - 性格通过剧情展现，不是描述

### 最实用的5条建议
1. 一章一高潮，全程无尿点
2. 删减无效剧情和描写
3. 快节奏开篇，不要只铺垫
4. 多看多写，扫榜学习
5. 每天进步一点点，坚持就是胜利

### 最常见的3个错误
1. ❌ 主线不清晰，剧情太散
2. ❌ 开头过度铺垫，没有吸引力
3. ❌ 人物性格只描述不展现
