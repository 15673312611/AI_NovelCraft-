# 小说大纲写作技巧（七猫编辑版）

来源：七猫中文网编辑团队

## 一、作品设定

### 1. 人物设置

人物设置包含主角，以及在作品中对剧情发展有重要推动作用的人物。这就与我们看电影时的主要演员及其配角一样。

**在设定主要人物时，应当包括：**
- 人物姓名
- 身世家境
- 性格特点
- 外貌喜好
- 力量属性

**重点：性格特点的设定**

在人物设置中，不论何种作品，其中最为重要的一个设定是"性格特点"。许多作品里面的人物，写出来像看木偶一样，一点也没有灵性，就是因为作者根本没把握好"性格特点"。

**如何让人物充满灵性：**
- 仔细设定一些受读者广泛喜爱的性格特点
- 在行文中时刻注意人物塑造是否符合该人物的性格设定
- 让每一个人物都活灵活现

**人物简介要点：**
- 设置出让读者眼前一亮的人物简介
- 突出人物的独特性和吸引力

### 2. 世界设置

世界设置对玄幻、网游类作品极为需要，这个设置即对故事所发生的世界进行一个框架的描述，以免自己在行文中，会让自己架构的世界变形。

**世界设定应当包括：**
- 世界组成
- 势力分布
- 力量体系
- 货币换算等

**网游文中的设定示例：**
- 世界地图
- 怪物分布
- 公会势力
- 角色职业
- 角色技能等

认真架构完善作品中的世界，对于完善故事具有极大的作用。

## 二、故事主线

故事主线，即作品故事发展的主要脉络。

**主线设定示例：**
- 学成归来的高手下山去复仇、退婚、祸害师姐
- 少年穿越古代娶妻纳妾、修身齐家、争霸天下
- 小人物重生后利用先知优势成为商界大亨的发迹史

**主线设定时需要思考：**
- 自己这个作品，究竟想用什么来吸引读者
- 想想究竟能不能吸引读者
- 这也是我们日常交流中所谓的**期待感**

## 三、分集大纲

分集大纲应当包括两个部分：

### 1. 本阶段出场人物

- 将该阶段中所有主要角色全部列出来
- 如果主要角色中有在"相关设置"中没有设定的非主角，也需要用简单的话语描述一下这个角色的性格

### 2. 本阶段故事详纲

本阶段故事详纲，是该阶段故事发展的一个详细介绍。

**需要包含三个要素：**
- 人物
- 地点
- 事件

即：某人在某地做了什么，发生了什么故事，然后结果又怎么样。

**分集故事详纲的设定建议：**
- 如果能在一开始便将整个作品所有分集设定出来最好
- 就算后有改动在写作时也能做到心理有谱
- 如果不能设定出来，也至少要设定出今后10万字左右的故事内容

**设定分集故事详纲时要注意：**
- 仔细考虑所设定的这一集故事中的情节，是否是在围绕着主线故事进行
- 是否是在推动着故事的发展

## 四、大纲写作的核心原则

1. **完整性**：大纲要涵盖整个故事的起承转合
2. **逻辑性**：情节发展要符合逻辑，不能前后矛盾
3. **灵活性**：大纲可以调整，但主线不能偏离
4. **可读性**：大纲要清晰明了，方便自己和编辑查看

## 五、常见问题

### 问题1：大纲写得太详细，写作时没有创作激情怎么办？

**解决方案：**
- 大纲只需要写到关键情节点
- 具体的对话、细节描写可以在写作时发挥
- 保留一定的创作空间

### 问题2：写作过程中发现大纲不合理怎么办？

**解决方案：**
- 及时调整大纲
- 但要确保调整后的情节不会影响主线
- 重大调整需要重新梳理后续情节

### 问题3：如何判断大纲是否合格？

**判断标准：**
- 主线清晰，目标明确
- 人物性格鲜明，行为符合逻辑
- 情节有起伏，有冲突有爽点
- 世界观完整，不自相矛盾

---

**编辑建议：**
认真写好大纲，是小说创作成功的第一步。一个好的大纲能让你的创作事半功倍，也能大大提高签约的成功率。
