# 小说大纲写作技巧

**来源：** 七猫中文网 - 编辑：开心

## 一、作品设定

### 1. 人物设置
人物设置包含主角，以及在作品中对剧情发展有重要推动作用的人物。

**人物设定应包括：**
- 人物姓名
- 身世家境
- 性格特点（最重要）
- 外貌喜好
- 力量属性
- 让读者眼前一亮的人物简介

**性格特点的重要性：**
- 许多作品里面的人物像木偶一样，没有灵性，就是因为作者没把握好"性格特点"
- 要想让人物充满灵性，活灵活现，就要仔细设定受读者广泛喜爱的性格特点
- 在行文中时刻注意，看自己对人物的塑造是否符合该人物的性格设定

### 2. 世界设置
玄幻、网游类作品极为需要世界设置，对故事所发生的世界进行框架描述。

**世界设定应包括：**
- 世界组成
- 势力分布
- 力量体系
- 货币换算

**类比网游文设定：**
- 世界地图
- 怪物分布
- 公会势力
- 角色职业
- 角色技能

## 二、故事主线

故事主线，即作品故事发展的主要脉络。

**主线示例：**
- 学成归来的高手下山去复仇、退婚、祸害师姐
- 少年穿越古代娶妻纳妾、修身齐家、争霸天下
- 小人物重生后利用先知优势成为商界大亨的发迹史

**设定主线时要思考：**
- 这个作品究竟想用什么来吸引读者
- 能不能吸引读者
- 期待感如何营造

## 三、分集大纲

分集大纲应当包括两个部分：

### 1. 本阶段出场人物
- 将该阶段中所有主要角色全部列出来
- 如果主要角色中有在"相关设置"中没有设定的非主角，也需要用简单的话语描述性格

### 2. 本阶段故事详纲
需要人物、地点和事件三个要素：
- 某人在某地做了什么
- 发生了什么故事
- 结果又怎么样

**分集故事详纲建议：**
- 如果能在一开始便将整个作品所有分集设定出来最好
- 至少要设定出今后10万字左右的故事内容
- 要仔细考虑，所设定的这一集故事中的情节，是否是在围绕着主线故事进行
- 是否是在推动着故事的发展

## 核心要点

1. **人物塑造** - 性格特点是关键，让人物活起来
2. **世界架构** - 完善的世界观支撑故事发展
3. **主线清晰** - 明确故事要讲什么，吸引点在哪
4. **分集规划** - 至少规划10万字内容，围绕主线推进
