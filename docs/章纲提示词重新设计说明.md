# 章纲提示词重新设计说明

**设计时间**：2024年12月11日
**设计目标**：结合提示词示例框架的设计技巧和七猫爆款的节奏技巧，重新设计章纲生成提示词

---

## 一、设计理念

### 1. 借鉴提示词示例框架的核心技巧

#### （1）角色定位清晰化
```
❌ 旧版：你是一名资深网文章纲策划编辑
✅ 新版：你是网文章纲首席架构师+剧情公式化设计专家，拥有十年以上网文策划经验
```

#### （2）使命明确化
```
❌ 旧版：让剧情推进清晰、有因有果
✅ 新版：
1. 设计N章章纲，每章都有明确的核心看点
2. 确保前3章快速建立冲突+人设记忆点+悬念链
3. 在第7-10章设计信息差爆发和打脸高潮
4. 所有[看点]和[收束]必须具体可视化
```

#### （3）三步处理流程
```
Step 1：深度剧情解析（2分钟思考）
Step 2：章节骨架构建（应用公式库）
Step 3：质量检查（每章自检）
```

### 2. 借鉴七猫爆款的节奏技巧

#### （1）前3章节奏
- 第1章：建立高信息量冲突（如：发现男友出轨+刷卡49998元）
- 第2章：引入生死赌注（如：奶奶癌症需要100万）
- 第3章：设计意图不明的行为（如：打电话，但不说目的）

#### （2）第4-6章节奏
- 第4章：部分揭晓（如：契约婚姻），但埋下新疑问（如：半年后会怎样）
- 第5章：意外打断（如：刘婶刁难）
- 第6章：信息差埋伏笔（如：盛明羲是盛通总裁，但女主不知道）

#### （3）第7-10章节奏
- 第7章：信息差爆发（如：盛明羲误会升级）
- 第8-9章：打脸高潮准备
- 第10章：卷末收束+埋下一卷伏笔

## 二、核心设计公式

### 公式 1：人设记忆点构建公式
```
角色魅力 = [反差行为模式] × [重复强化次数]
```

**四大模板**：
- 模板 1-A：反差人设
- 模板 1-B：能力碾压
- 模板 1-C：隐藏底牌
- 模板 1-D：当场反击

### 公式 2：悬念链构建公式
```
读者粘性 = [信息差层数] × [揭晓节奏控制]
```

**三大模板**：
- 模板 2-A：信息揭示循环
- 模板 2-B：信息差分层
- 模板 2-C：打脸爽点

### 公式 3：冲突升级公式
```
冲突强度 = [赌注价值] × [不可逆程度]
冲突层级：情绪冲突 < 利益冲突 < 生死冲突
```

### 公式 4：Show, Don't Tell
```
角色状态 = [可观察的外部行为] - [内心独白]
```

## 三、关键改进点

### 1. [看点] 标签强化

**旧版**：
```
[看点] 奢华压抑的家宴上，姜月初的精心准备被彻底无视，凸显其在沈家的工具人地位。
```
→ 抽象、说明性

**新版要求**：
```
必须包含：[具体动作] + [对手反应] + [冲击效果]
检查标准：能否拍成一个3秒的短视频画面？
```

**期望效果**（参考七猫爆款）：
```
[看点] 当林硕和秦涵月以为陆清清只能净身出户时，她拿出房本，原来这房子是她的。林硕和秦涵月当场傻眼。
```

### 2. [收束] 标签强化

**旧版**：
```
[收束] 晚宴结束，姜月初看着桌上未被动过的长寿面，平静地将其倒掉。
```
→ 虽然是外显行为，但缺少"反差冲击"

**新版要求**：
```
必须包含：[角色A的动作] + [可观察的状态变化]
禁止词："她想"、"她感到"、"她意识到"、"她决定"
检查标准：能否用摄像机拍下来？
```

**期望效果**（参考七猫爆款）：
```
[收束] 陆清清拿出房本的瞬间，秦涵月和林硕的表情从得意变成震惊，然后灰溜溜离开。
```

### 3. 增加设计步骤指导

```
【如何设计具体的[看点]】：
步骤1：确定本章要使用哪个模板（1-A/1-B/1-C/1-D/2-C）
步骤2：填充变量（如：X=被羞辱的程度，Y=实力差距）
步骤3：设计具体动作（拿出什么？说了什么？做了什么？）
步骤4：设计对手反应（表情如何变化？说了什么？做了什么？）
步骤5：检查：能否拍成3秒短视频？如果不能，回到步骤3重新设计
```

## 四、与旧版的核心区别

| 维度 | 旧版 | 新版 |
|------|------|------|
| 角色定位 | 资深编辑 | 首席架构师+公式化设计专家 |
| 处理流程 | 无明确流程 | 三步处理流程（解析→构建→检查） |
| 公式设计 | 有公式，但缺少填充指导 | 公式+模板+填充步骤 |
| 检查标准 | 抽象（"可视化"） | 具体（"能否拍成3秒短视频"） |
| 节奏设计 | 三幕结构（前中后） | 分段节奏（1-3/4-6/7-10章） |
| 示例引用 | 无 | 参考七猫爆款的具体案例 |

## 五、预期效果

### 1. [看点] 质量提升
- ❌ 避免："凸显工具人地位"、"展示能力"等抽象描述
- ✅ 实现："她拿出房本，林硕和秦涵月当场傻眼"等具体画面

### 2. [收束] 质量提升
- ❌ 避免："她想"、"她感到"等内心独白
- ✅ 实现："她删除了他的联系方式"等外显行为

### 3. 悬念链质量提升
- ❌ 避免：第3章就说"我要离婚"
- ✅ 实现：第3章只说"她给一个陌生号码打了电话"

### 4. 冲突赌注质量提升
- ❌ 避免：只有"情绪冲突"或"面子"
- ✅ 实现：具体的"49998元+房子归属"等利益冲突
