# 微信登录配置指南

## 概述

本系统支持两种微信登录方式：

| 方式 | 适用场景 | 要求 | 用户体验 |
|------|----------|------|----------|
| **开放平台（PC）** | PC网页 | 企业资质 | 扫码登录 |
| **公众号（H5）** | 微信内网页 | 认证服务号 | 自动授权 |

你可以根据需要启用其中一种或两种都启用。

---

## 方式一：微信公众号登录（推荐）

**优点**：门槛低，只需要认证的服务号即可

### 准备工作

1. **拥有认证的微信服务号**（订阅号不支持网页授权）
   - 如果没有，可以在 [微信公众平台](https://mp.weixin.qq.com/) 注册

2. **获取 AppID 和 AppSecret**
   - 登录公众平台 → 开发 → 基本配置
   - 记录 AppID（开发者ID）
   - 重置并记录 AppSecret（开发者密码）

3. **配置网页授权域名**
   - 设置 → 公众号设置 → 功能设置 → 网页授权域名
   - 填写你的网站域名（不带 http://）
   - 需要下载验证文件放到网站根目录

### 系统配置

1. 登录 Admin 后台
2. 进入「微信登录配置」
3. 切换到「Service Account (H5)」标签
4. 填写：
   - Enable H5 Login: 开启
   - Service Account AppID: 你的公众号 AppID
   - Service Account AppSecret: 你的公众号 AppSecret
5. 填写回调地址：`https://你的域名/login`
6. 保存配置

### 使用场景

用户在微信内打开你的网站时，点击登录会自动跳转授权页面，授权后自动登录。

---

## 方式二：微信开放平台登录

**优点**：支持PC网页扫码登录

### 准备工作

1. **注册微信开放平台账号**
   - 访问 [微信开放平台](https://open.weixin.qq.com/)
   - 需要企业资质认证

2. **创建网站应用**
   - 管理中心 → 网站应用 → 创建网站应用
   - 填写应用信息，提交审核

3. **获取凭证**
   - 审核通过后获取 AppID 和 AppSecret

4. **配置授权回调域**
   - 应用详情 → 开发信息 → 授权回调域
   - 填写你的网站域名

### 系统配置

1. 登录 Admin 后台
2. 进入「微信登录配置」
3. 在「Open Platform (PC)」标签下填写：
   - Enable PC Login: 开启
   - Open Platform AppID
   - Open Platform AppSecret
4. 填写回调地址
5. 保存配置


## 回调地址说明

回调地址是微信授权成功后重定向的地址。

格式：`https://你的域名/login`

**注意事项**：
- 必须使用 HTTPS
- 域名必须与微信后台配置一致
- 不要添加额外路径或参数

## 登录流程

### PC扫码登录
1. 用户点击「微信登录」
2. 显示微信二维码
3. 用户用微信扫码确认
4. 自动登录/注册

### H5授权登录（微信内）
1. 用户在微信内打开网站
2. 点击「微信登录」
3. 跳转微信授权页面
4. 用户点击「同意」
5. 自动登录/注册

---

## 新用户处理

- 首次微信登录自动创建账户
- 用户名：`wx_xxxxxxxx`（随机）
- 昵称和头像从微信获取
- 自动赠送初始灵感点

## 常见问题

### Q: 公众号和开放平台有什么区别？

| 对比项 | 公众号（服务号） | 开放平台 |
|--------|------------------|----------|
| 资质要求 | 认证服务号 | 企业资质 |
| 适用场景 | 微信内H5 | PC网页 |
| 用户体验 | 自动授权 | 扫码登录 |
| 申请难度 | 较低 | 较高 |

### Q: 可以同时启用两种吗？
可以。系统会自动判断：
- 在微信内打开 → 使用公众号授权
- 在PC浏览器 → 使用开放平台扫码

### Q: 订阅号可以吗？
不可以。网页授权需要认证的服务号。

### Q: 本地如何测试？
微信登录需要公网域名，本地测试可以：
1. 使用 ngrok 等内网穿透工具
2. 配置测试号（公众平台提供）

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /auth/wechat/config | GET | 获取配置 |
| /auth/wechat/auth-url?type=open\|mp | GET | 获取授权URL |
| /auth/wechat/login | POST | 登录 |
| /auth/wechat/bind | POST | 绑定微信 |
| /auth/wechat/unbind | POST | 解绑微信 |
