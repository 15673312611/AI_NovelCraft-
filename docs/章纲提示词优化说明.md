# 章纲生成提示词优化说明

## 优化时间
2024年12月11日

## 问题诊断

根据编辑反馈，原章纲存在以下核心问题：

### 1. 剧情推进过于线性和预期内
- **问题**：受虐→觉醒→离婚→独立，每一步都在读者预期之内
- **后果**：读者能提前预判后续5章走向，缺乏阅读惊喜

### 2. 冲突烈度不足
- **问题**：只有"婆婆骂人→女主走人"这种情绪冲突
- **后果**：没有利益冲突或生死冲突，赌注太低，不够抓人

### 3. 人设记忆点严重不足
- **问题**：标签只停留在叙述层面（"医学天才"只通过旁白说明）
- **后果**：没有通过具体行为让读者记住角色特点

### 4. 看点设计流于表面
- **问题**：反击太隐蔽（如"删除文件"），金手指只"说"不"演"
- **后果**：读者感知不到爽点，缺乏视觉冲击力

## 优化方案

### 核心修改点

#### 1. 增加"意料之外、情理之中"的转折要求
```
- 让剧情推进清晰、有因有果，但要在关键节点设计"意料之外、情理之中"的转折
- 确保每章都有明确的"核心看点"，避免平均摊开多个平淡小事
```

#### 2. 强化卷首三章的外显行为要求
```
- 第1-2章的反击必须是【读者可感知的外显行为】，而不是内心独白或隐蔽动作（如删除文件）
- 第1-3章的金手指展示必须通过【具体场景和行为】呈现，而不是旁白叙述或律师转述
```

#### 3. 新增"避免常见问题"专项指导
针对编辑反馈的高频问题，提供具体的"禁止"和"改进"对照：

**剧情过于线性可预测**
- ❌ 禁止：受虐→觉醒→离婚→独立 这种完全在预期内的流程
- ✅ 改进：在关键节点设计"意外转折"（离婚当天发现怀孕/被诬陷偷窃/发现丈夫窃取研究成果等）

**冲突烈度不足**
- ❌ 禁止：只有"婆婆骂人→女主走人"这种低烈度情绪冲突
- ✅ 改进：增加利益冲突（专利纠纷/财产争夺）或生死冲突（疾病/陷害），让赌注更高

**人设记忆点弱**
- ❌ 禁止：只在旁白中说"她是医学天才"，或通过律师转述"她有研究能力"
- ✅ 改进：通过具体场景展示（用医学知识救人/识破药物造假/推翻权威专家错误诊断）

**看点设计流于表面**
- ❌ 禁止："删除文件"这种读者感知不到的隐蔽反击
- ✅ 改进：设计外显的、有视觉冲击力的反击（当众揭露真相/拒绝签字并提出反诉/用专业能力让对方当场出丑）

#### 4. 强化关键标签的具体要求

**[冲突]标签**
```
冲突必须有明确的利害关系（情绪冲突<利益冲突<生死冲突），
避免只写"婆婆骂女主"这种低烈度冲突。
优先设计：专利/财产/职位/名誉/健康等实质性赌注。
```

**[看点]标签**
```
看点必须具体、可视化、有冲击力，
避免抽象描述（如"女主内心坚定"应改为"女主当众撕毁协议并提出反诉"）。
```

## 预期效果

通过以上优化，生成的章纲应该能够：

1. **提升剧情辨识度**：在关键节点设计意外转折，避免完全线性推进
2. **增强冲突张力**：从纯情绪冲突升级到利益/生死冲突，提高赌注
3. **强化人设记忆点**：通过具体场景和行为展示角色特质，而非旁白叙述
4. **优化看点设计**：确保每章都有外显的、可视化的核心看点

## 使用建议

1. 生成章纲后，重点检查前3章是否有：
   - 至少1个"意料之外"的转折
   - 至少1次外显的反击行为
   - 至少1个具体场景展示金手指

2. 检查每章的[冲突]是否有实质性赌注（不只是情绪冲突）

3. 检查每章的[看点]是否具体可视化（不是抽象描述）
