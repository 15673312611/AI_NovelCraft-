# 章纲优化前后对比示例

## 示例场景：离婚复仇文开篇

### ❌ 优化前（编辑评价：剧情平淡、看点不足）

#### 第1章
```json
{

    "[冲突] 姜月初 vs 沈修谨+婆婆",
    "[看点] 极致虐心开局：丈夫当众牺牲她来保护大嫂",
    "[结果] 姜月初内心一片死灰，眼神变得异常平静",
    "[收束] 姜月初在内心对自己说：'沈修谨，我们结束了'"
  ]
}
```

**问题诊断**：
- 冲突只是"婆婆指责"，烈度太低
- 看点是"内心死灰"，读者感知不到
- 收束是"内心独白"，没有外显行为

#### 第2章

- "删除文件"是隐蔽反击，读者感知不到爽点
- 全章都是回忆和内心戏，缺乏外显冲突
- 没有展示女主的金手指能力

**问题诊断**：
- 金手指只是"展示摘要"，没有具体场景

- 剧情完全在预期内（联系律师→起草协议）

### ✅ 优化后（针对性改进）

#### 第1章
```json
{
  "chapterInVolume": 1,
  "keyPlotPoints": [
    "[事件] 沈修谨大哥的葬礼上，姜月初被婆婆推搡时，意外听到婆婆和林婉柔的私语：'还好月初那个蠢货，把她导师的癌症药方案都给了修谨...'",
    "[冲突] 姜月初 vs 沈修谨+婆婆+林婉柔。认知颠覆：不是'不够好被当替身'，而是'从一开始就是被利用窃取研究成果的工具人'。赌注：价值数十亿的专利归属权。",
    "[看点] 意外转折：她想起沈修谨公司最近上市的'曦光1号'抗癌药，正是基于她三年前交给他的研究方案。这不是感情问题，是知识产权盗窃。",
    "[推进] 【主线】从'婚姻不幸'升级为'商业欺诈+知识产权纠纷'，冲突烈度质的飞跃。",
    "[结果] 姜月初没有哭闹，而是冷静地用手机录下了这段对话，作为日后诉讼的证据。",
    "[收束] EMOTIONAL_BEAT：她平静地走出灵堂，眼神从绝望变为锋利。这不是离婚，是战争。",
    "[钩子] STRONG_HOOK：她拨通了一个尘封三年的电话号码——她的导师，国际医学专利律师团队的首席顾问。"
  ]
}
```

**改进点**：
- ✅ 增加"意外转折"：从婚姻问题升级为知识产权盗窃
- ✅ 提升冲突烈度：从情绪冲突→利益冲突（数十亿专利）
- ✅ 外显行为：录音取证（而非内心独白）
- ✅ 强钩子：联系导师律师团队

## 对比总结

| 维度 | 优化前 | 优化后 |
|------|--------|--------|
| **剧情可预测性** | 完全线性：受虐→觉醒→离婚 | 有意外转折：发现知识产权盗窃 |
| **冲突烈度** | 低（婆婆骂人） | 高（数十亿专利纠纷+公开对峙） |
| **人设展示** | 旁白叙述（"她是医学天才"） | 具体场景（当场推翻律师判断+公开质疑技术细节） |
| **看点可感知性** | 隐蔽（删除文件、内心独白） | 外显（录音取证、当众扔起诉书） |
| **读者情绪** | 压抑、平淡 | 爽快、期待后续 |

## 关键改进策略

1. **转折设计**：在第1章就埋入"知识产权盗窃"这个意外信息，让剧情不只是婚姻问题
2. **冲突升级**：从纯情绪冲突→利益冲突（专利+股价）
3. **能力展示**：通过"推翻律师判断"和"公开质疑技术细节"两个具体场景，让读者直观感受女主的专业能力
4. **外显爽点**：用"当众扔起诉书"替代"删除文件"，让反击可视化
5. **实质后果**：增加"股价下跌8%"这种可量化的影响，提升冲突真实感
