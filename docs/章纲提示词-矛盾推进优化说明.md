# 章纲提示词-矛盾推进优化说明

**优化时间**：2024年12月11日 16:45
**优化目标**：基于抖音博主的"矛盾推进剧情"理论，重新优化章纲提示词

---

## 一、核心问题

### 1. 生成的[看点]还是太复杂

**生成的章纲** ❌：
```
[看点] 沈修谨将白月紧紧护在身后，对自己的母亲怒吼"妈，够了！"，镜头扫过一旁被彻底无视的姜月初，她手里还端着刚切好的蛋糕，脸上一瞬间的错愕转为死寂般的平静。
```
→ 太长、太多细节、有"镜头扫过"这种导演视角

**七猫爆款** ✅：
```
[看点] 当林硕以为陆清清只能净身出户时，她拿出房本。林硕当场傻眼。
```
→ 简单、直接、一句话说清

### 2. 缺少"矛盾→困境→期待"的核心逻辑

博主的核心观点：
- **矛盾 = 冲突 = 人与人/人与环境的对立**
- 通过矛盾创造【困境】→ 引发读者【期待】→ 主角【行动】→ 推进剧情

---

## 二、优化内容

### 1. 加入"矛盾推进剧情"的核心创作理论

```
【核心创作理论：矛盾推进剧情】

矛盾 = 冲突 = 人与人/人与环境的对立
通过矛盾创造【困境】→ 引发读者【期待】→ 主角【行动】→ 推进剧情

五大矛盾类型：
1. 身份矛盾：穿越成逃犯/穷困潦倒的身份/被追杀的修士 → 困境：活下去
2. 环境矛盾：世界末日/生化危机/全民游戏 → 困境：适应环境
3. 生死危机矛盾：打了小的来老的/被陷害/被追杀 → 困境：保命
4. 阶层矛盾：泥腿子想要提升阶级/寒门想入豪门 → 困境：突破阶层
5. 感情矛盾：爱而不得/得而不爱/修罗场 → 困境：情感纠葛
```

### 2. 新增"公式1：矛盾设计公式"（最重要）

```
公式 1：矛盾设计公式（最重要）
每章必须有1个核心矛盾 = [主角当前困境] × [解决困境的难度]

模板 1-矛盾设计三步法：
步骤1：设定困境（主角面临什么问题？活不下去/被欺负/被误会/被追杀？）
步骤2：制造期待（读者想看什么？主角怎么反击/怎么逃脱/怎么证明？）
步骤3：设计行动（主角做什么具体动作？拿出底牌/当场反击/巧妙化解？）
```

### 3. 简化[看点]要求

**旧版**：
```
必须包含：[具体动作] + [对手反应] + [冲击效果]
检查标准：能否拍成一个3秒的短视频画面？
```

**新版**：
```
格式：用1句话说清楚"谁做了什么+对手的反应"（20-30字）

❌ 禁止写法：
- 太长（超过50字）
- 有导演视角（"镜头扫过"、"特写"）
- 抽象描述（"凸显工具人地位"、"展示能力"）

✅ 正确写法（参考七猫爆款）：
"当林硕以为陆清清只能净身出户时，她拿出房本。林硕当场傻眼。"
"刘婶要求陆清清做饭还要分工资，陆清清直接说'分我一半，否则我给盛明羲打电话'。刘婶哑口无言。"

检查标准：能否用一句大白话复述？如果不能，重新设计。
```

### 4. 调整公式编号

- **公式 1**：矛盾设计公式（新增）
- **公式 2**：人设记忆点构建公式
- **公式 3**：悬念链构建公式
- **公式 4**：冲突升级公式
- **公式 5**：Show, Don't Tell

## 三、核心改进逻辑

### 旧版问题
1. [看点]太复杂，有导演视角
2. 缺少"矛盾→困境→期待→行动"的循环逻辑
3. 没有明确每章必须有1个核心矛盾

### 新版解决方案
1. **矛盾为核心**：每章必须有1个核心矛盾，通过三步法设计
2. **[看点]简化**：用1句大白话说清楚，参考七猫爆款
3. **五大矛盾类型**：给AI明确的矛盾类型库，方便选择

## 四、参考案例（来自抖音博主）

### 案例1：《一世之尊》
- **身份矛盾**：主角身份卷入超顶尖战力的布局，自带天然使命
- **核心逻辑**：为了摆脱命运，不断挣扎，反抗自身宿命

### 案例2：《诡秘之主》
- **扮演法**：升级存在风险，必须做特定事情（给支线剧情理由）
- **序列唯一性**：只要升级，就触碰既得利益者利益（自动制造矛盾）

### 案例3：《玄鉴仙族》《苟在魔门》
- **果位唯一性**：你只要前行，就会触碰其他人利益
- **上位者布局**：全文都是坑，只有一个个淌过去才能解决终极矛盾

### 案例4：《遮天》
- **理念矛盾**：理想主义 vs 现实，道争，追求成帝
- **人物弧光**：通过理念追求，塑造强烈的人设共鸣

## 五、预期效果

### 1. [看点]简化
- ❌ 避免："沈修谨将白月紧紧护在身后，对自己的母亲怒吼，镜头扫过姜月初..."（太长、有导演视角）
- ✅ 实现："当沈修谨要护白月时，姜月初平静地放下蛋糕，转身回房。"（简单、直接）

### 2. 矛盾设计清晰
- 每章都有明确的【困境】→【期待】→【行动】循环
- AI知道该从哪个矛盾类型入手（身份/环境/生死危机/阶层/感情）

### 3. 剧情推进有逻辑
- 通过矛盾推进剧情，而不是堆砌事件
- 每个困境都引发读者期待，每个行动都推进剧情

---

## 六、关键优化点总结

1. **新增"矛盾推进剧情"核心理论**
2. **新增"公式1：矛盾设计公式"（最重要）**
3. **简化[看点]要求**：用1句大白话，禁止导演视角
4. **提供五大矛盾类型库**
5. **强调"困境→期待→行动"的循环**

---

**总结**：新版提示词采用"矛盾驱动"的设计理念，参考网文大神的创作经验（《一世之尊》《诡秘之主》《遮天》等），确保每章都有明确的矛盾冲突和剧情推进逻辑。
