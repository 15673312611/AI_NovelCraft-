# 章纲弹窗美化更新说明

## 更新概述
对 WritingStudioPage 中的章纲生成弹窗和章纲缺失提醒弹窗进行了全面的视觉美化升级。

## 主要改进

### 1. 章纲生成弹窗 (Outline Generate Modal)

#### 视觉增强
- **更大的图标区域**: 从 64x64 增大到 72x72，视觉冲击力更强
- **增强的阴影效果**: 多层阴影 + 发光边框，立体感更强
- **动态脉冲动画**: 生成时的呼吸动画效果，配合闪光效果
- **更精致的圆角**: 从 10px 升级到 11px
- **优化的间距**: padding 从 24px 增加到 32px

#### 交互优化
- **按钮悬停效果**: 
  - 取消按钮: 平滑的颜色过渡和轻微上移
  - 生成按钮: 更明显的阴影增强和位移效果
- **鼠标交互反馈**: 所有按钮都有 `onMouseEnter` 和 `onMouseLeave` 事件
- **禁用状态优化**: 生成中时按钮状态清晰

#### 卷信息卡片
- **更大的卷号徽章**: 从 40x40 增加到 48x48
- **光泽效果**: 徽章内添加高光装饰
- **更好的布局**: flex: 1 确保文本区域充分利用空间
- **小装饰点**: 章节范围前添加装饰性圆点

#### 进度显示
- **更粗的进度条**: strokeWidth 从默认增加到 8
- **进度信息显示**: 百分比和预计时间分布在进度条两侧
- **状态卡片**: 蓝色渐变背景的状态提示卡
- **三色渐变**: 进度条使用 0% → 50% → 100% 的三色渐变

#### 说明文本
- **左侧装饰条**: 4px 宽的紫色渐变竖条
- **强调文字**: "详细的写作指导" 使用 strong 标签加粗
- **更好的行高**: lineHeight 1.7 提升可读性

### 2. 章纲缺失提醒弹窗 (Outline Missing Modal)

#### 视觉风格
- **警告色系**: 橙黄色主题 (#fa8c16, #f59e0b)
- **温暖渐变背景**: #fffbeb → #fef3c7
- **警告装饰条**: 左侧橙色渐变竖条
- **高光装饰**: 图标区域添加白色光晕效果

#### 按钮布局
- **三按钮设计**: 取消 / 生成章纲 / 继续生成
- **尺寸一致性**: 所有按钮统一 44px 高度
- **智能宽度**: 根据按钮重要性设置不同 minWidth

### 3. CSS 动画

添加了三个关键帧动画：

```css
@keyframes pulse
- 图标呼吸效果
- scale(1) → scale(1.03) → scale(1)
- 配合阴影变化

@keyframes shimmer  
- 闪光扫过效果
- 对角线移动的光带

@keyframes fadeIn
- 淡入效果
- opacity + translateY
```

### 4. 弹窗容器优化

```css
.outline-generate-modal .ant-modal-content
.outline-missing-modal .ant-modal-content
- 圆角: 16px
- 阴影: 双层阴影增强立体感
- 关闭按钮: 40x40, 圆角 10px, 悬停背景色
```

## 技术细节

### 颜色方案
- **主色**: #667eea → #764ba2 (紫色渐变)
- **蓝色**: #1890ff, #e6f7ff (进度状态)
- **橙色**: #fa8c16, #fffbeb (警告提示)
- **中性色**: #64748b, #94a3b8 (次要文本)

### 动画曲线
- `cubic-bezier(0.4, 0, 0.2, 1)`: 标准缓动
- `ease-in-out`: 脉冲动画
- `linear`: 闪光动画

### 响应式设计
- 弹窗宽度: 520px (缺失) / 560px (生成)
- 自适应内容高度
- 移动端友好的 centered 布局
